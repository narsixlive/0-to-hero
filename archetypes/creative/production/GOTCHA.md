# Gotchas — Production

## Critical (always read)
- ❌ Delivering without checklist → ✅ Checklist mandatory before any send (2026-02-15)
  Delivery checklist:
  - [ ] Correct format (WAV 48kHz/24bit)
  - [ ] Stems included if sync client
  - [ ] Naming compliant: [client]_[title]_FINAL_[date].wav
  - [ ] Final listen on headphones + speakers
  - [ ] Summary email written
- ❌ Delivering from the work file → ✅ Always export from a clean session (2026-03-02)
